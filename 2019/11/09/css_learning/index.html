css学习笔记

一、css简介

  1. 指层叠样式表（cascading style sheets）
  2. 样式优先级：浏览器自定义 < 外部样式表（头文件中<link>标签可以实现）< 内部样式表（头文件中<style>标签可以实现）< 内联样式（在元素内部的style属性）

二、css样式表

​ 通过选择器定义样式的作用标签，通过声明确定要添加的属性

​ 内部样式表/外部样式表语法： 选择器 { 声明；声明；… }

​ 内联样式语法： style=” 声明；声明；”

（一）选择器：

  1. 分组选择器：所有同类标签都作用（逗号符）

    语法： 标签（，标签，标签…）

    如： h1，h2，h3

    （注意继承问题：例子中h1，h2，h3标签的后代标签也会被定义）

  1. 后代选择器：作用于指定标签的任意后代标签（空格符）

    语法： 标签 [（后代）标签 （后代）标签 … ]

    如： span p strong

    (注意，例子中属性会作用于 <span><p><li><strong></strong></li></p></span>中的<strong> 标签)

  1. 子元素选择器：只作用于指定标签的子标签（大于符）

    语法： 标签 > 子标签

    如： h1 > strong

  1. 兄弟元素选择器：作用于指定标签的有同一个父标签的兄弟标签（加号符）

    语法： 标签 + （兄弟）标签

    如： ul li + li

    （例子中属性会作用于ul元素的后代元素中第一个li后所有的li，如果再加一个li则从第三个开始）

  1. 属性选择器：作用于含有某个属性或者含有某个属性且属性值一定的标签

    语法： [属性] 或 [属性（记号）=属性值]

    ~= 属性值含有空格分出的指定值 [title~=hello],则title="helloworld"不可以
    |= 属性值以指定值开头且必须后面没有值或后面的值用-分开 [title|=hello],则title="hello-world"可以
    ^= 属性值以指定值开头即可 [title^=hello],则title="hellowor"可以
    $= 属性值以指定值结尾 [title$=hello],则title="safdhello"可以
    *= 属性值中存在指定值 [title*=hello],则title="helloworld"可以

  1. id选择器：作用于id值为指定值的标签

    语法： #指定值

    例： (span)#intro

    （例子中如果加span则指id属性值为intro的所有span标签被定义；之前的1~4中选择器也可以叠加使用）

  1. class选择器：作用于class值含有指定值的标签

    语法： .指定值（.指定值） （有多个指定值时满足每一个才被定义）

    例子 span.high.weigh

  2. *选择器：使样式作用于所有标签

（二）声明：

​ 属性：属性值 ;（属性：属性值 ; … ）

三、css常用属性

（一）样式类

  1. 背景：

    1. background-color属性：设置标签的背景颜色

      属性值：颜色名、十六进制颜色值、RGB

    2. background-image属性：设置标签的背景图像

      属性值：url（绝对地址/相对地址）

      例：background-image: url(/i/eg_bg_03.gif)

    3. background-repeat属性：设置背景图像的重复方式

      属性值 效果
      repeat 图像在水平垂直方向上都平铺（也是未设置background-repeat属性的默认效果）
      repeat-y 图像在垂直方向上平铺
      repeat-x 图像在水平方向上平铺
      no-repeat 不平铺

    4. background-position属性：设置背景图像在标签中的位置（默认为top left）

      属性值 效果
      关键字（两两结合可以实现九个位置，只有一个则视为水平值） top 顶部
      center 中间
      bottom 底下
      left 左边
      right 右边
      百分数值（只有一个则视为水平值）（例：background-position：33% 66%） 例子的效果是图像的中心点在水平离父元素左边界三分之一的地方，竖直离上边界三分之二的地方
      长度值（只有一个则视为水平值）（例：background-position：50px 100px） 例子的效果是图像的左上角在水平离父元素左边界50个像素点的地方，竖直离上边界100个像素点的地方

    5. background属性：可以直接将以上的所有属性值用空格分开然后全部写在background属性里面

      例：background: #ff0000 url(/i/eg_bg_03.gif) no-repeat fixed center

  2. 文本：

    1. color属性：设置字体颜色

    2. text-indent属性：首行缩进(注意，可以继承)

      属性值：数值（可为负）+ em，cm，px … 或 百分比

    3. text-align属性：水平对齐

      属性值 效果
      left 左对齐
      right 右对齐
      center 居中
      justify 两端对齐
      inherit 从父元素中继承

    4. word-spacing属性：改变单词之间的间隔

      属性值：数值（可为负）+ em，cm，px … （默认值为0）

    5. letter-spacing属性：改变单个字母之间的间隔

      属性值：同上

    6. text-transform属性：改变字母大小写

      属性值 效果
      none 默认值，不做任何改变
      uppercase 全大写
      lowercase 全小写
      capitalize 每个单词首字母大写

    7. text-decoration属性：装饰文字

      属性值 效果
      none 无装饰
      underline 下划线
      overline 上划线
      line-through 贯穿线
      blink 文本闪烁
      多个效果（例：text-decoration:underline overline；） 下划线

    8. white-space属性：告诉浏览器如何处理空白符

      属性值 效果
      normal 跟平常一样去掉多余空白符，自动换行
      pre 保留所有空白符且不自动换行
      nowrap 去掉多余空白符但不自动换行
      pre-wrap 保留所有空白符，自动换行
      pre-line 去掉多余空格符，保留换行符，自动换行

    9. direction属性：控制文本方向

      属性值：rtl 从右到左；ltr 从左到右

    10. text属性：可以直接将以上的所有属性值用空格分开然后全部写在text属性里面

  3. 字体：

    1. font-family属性：确定文本字体

      属性值：字体名（如果字体名包括特殊字符或空格，则需要用引号括起来，且要于属性本身用的那种引号不同）

    2. font-style属性：确定字体风格

      属性值 效果
      normal 默认值，正常显示
      italic 文本斜体显示
      oblique 文本倾斜显示

      italic 和 oblique 其实差不多

    3. font-variant属性：设置小型大写字母

      属性值：small-caps

    4. font-weight：设置文本粗细

      属性值 效果
      100~900 指定了九级加粗
      normal 正常大小（等于400）
      bord 加粗（等于700）
      border 比父元素粗一个等级
      lighter 比父元素细一个等级

    5. font-size：设置字体大小

      属性值：数值+ em，cm，px … 或 百分比

    6. font：可以直接将以上的所有属性值用空格分开然后全部写在font属性里面

  4. 链接：

    链接分为不同状态，可以调整不同状态下的链接样式：

    选择器 内容
    a:link 未被访问的链接
    a:visited 用户已访问的链接
    a:hover 鼠标指针位于链接的上方
    a:active 链接被点击的时刻

    • 之前的1~4中选择器也可以叠加使用；

    • a:hover 必须位于 a:link 和 a:visited 之后；a:active 必须位于 a:hover 之后；

    • 可以通过改变链接的样式背景颜色来形成类似按钮的效果；或者添加display属性

  5. 表格：

    1. border属性：设置表格的样式（table标签中的样式不会遗传给th标签和td标签）

      语法： border： 表格边框厚度 边框样式 边框颜色（可以不按顺序）

      例： border： 1px solid blue

      （border-style属性可以直接设置边框样式；border-color属性可以直接设置颜色；border-width属性可以直接设置边框厚度）

    2. border-collapse属性：设置折叠边框

      属性值：当为collapse时折叠边框

    3. width和height属性：设置宽度和高度

      属性值：数值+ em，cm，px … 或 百分比

    4. text-align属性：设置水平对齐方式

    5. vertical-align属性：设置垂直对齐方式

    6. caption-side属性：表格中标题的位置

    7. empty-cells属性：是否显示表格中的空单元格

      属性值 样式
      hide 隐藏表格中空单元格上的边框和背景
      show 默认值，显示空单元格上的边框和背景
      inherit 从父元素继承 empty-cells 属性的值

    8. table-layout属性：设置显示单元、行和列的算法

      属性值 样式
      automatic 默认。列宽度由单元格内容设定
      fixed 列宽由表格宽度和列宽度设定
      inherit 从父元素继承table-layout属性的值

  6. 轮廓：

    1. border属性：设置标签的边框样式（同表格的border属性）

    2. outline属性：设置标签的轮廓样式，其中：

      • outline-color设置颜色

      • outline-style设置样式，属性值

      • outline-width设置宽度

        属性值 效果
        关键字（两两结合可以实现九个位置，只有一个则视为水平值） thin 细轮廓
        medium 默认值，中等轮廓
        thick 粗轮廓
        百分数值或确定的值 按输入的值定
        inherit 从父元素继承轮廓宽度

（二）框模型

  1. 概述：
↕margin(外边距)
→ height (高度) ↑ padding(内边距) ↓
element （元素）
↑width(宽度) ↓border(边框)

标签是有物理格式的。图中“元素”是标签内容占的框，这个框的高度是height，宽度是width；这个框到“边框”的距离为内边距，标签的背景也呈现在边框内；“边框”外还有一个框，它到“边框”的距离就是外边距，外边框默认透明。margin属性和padding属性一般为零；外边距可以为负但内边框不能为负值

  1. padding属性：属性值：数值+ em，cm，px … 或 百分比（默认值为0）

    1. 属性值只有一个时：上下左右都被定义

    2. 属性值有两个时：第一个定义于上下，第二个定义于左右

    3. 属性值有三个时：第一个定义于上，第二个定义于左右，第三个定义于下

    4. 属性值有四个时：按 上 右 左 下 的顺序定义

      （直接设置可用padding-top属性、padding-right属性、padding-bottom属性和padding-left属性）

  2. border属性：包括宽度、样式和颜色

    1. border-style属性、border-color属性和border-width属性都允许多值，定义位置见padding属性的介绍
    2. 想单独设置可以在border-style属性、border-color属性、border-width属性后加-left、-right、-top、-bottom

  3. margin属性：数值（可为负值）+ em，cm，px … 或 百分比（默认值为0）（基本同上）

    注意外边距合并的情况
